# Protocol Converter

> 详细文档请见：[protocol_converter_pkg/README.md](protocol_converter_pkg/README.md)

协议转换器 — 支持 **OpenAI Chat Completions**、**OpenAI Responses** 和 **Anthropic Messages** 三种 API 协议的相互转换。

**当前版本**：v1.37.0

## 快速开始

```bash
cd protocol_converter_pkg
pip install -e .
```

详见 [完整文档](protocol_converter_pkg/README.md)。
